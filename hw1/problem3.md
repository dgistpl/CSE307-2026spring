# Problem 3: `suml : int list list -> int`

**Homework 1 - CSE307, Spring 2026**

## Description

Write a function

```
suml : int list list -> int
```

which takes a list of lists of integers and sums the integers included in all the lists.

## Examples

```ocaml
suml [[1;2;3]; []; [-1; 5; 2]; [7]] = 19
```
