## Overview

This assignment covers basic OCaml programming. You will implement 15 functions that exercise fundamental OCaml concepts including recursion, pattern matching, higher-order functions, algebraic data types, and inductive definitions.

## Instructions

- Each problem has its own `.ml` file (e.g., `problem1.ml`, `problem2.ml`, ...).
- Each file contains a function stub marked with `failwith "TODO"`. Replace the stub with your implementation.
- Test your solutions in `utop` by running:
  ```
  #use "hw1/problem1.ml";;
  ```
- You can also run:
  ```
  ocaml hw1/problem1.ml
  ```
