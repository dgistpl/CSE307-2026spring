# Problem 1: `prime : int -> bool`

**Homework 1 - CSE307, Spring 2026**

## Description

Write a function

```
prime : int -> bool
```

that checks whether a number is prime (`n` is prime if and only if `n` is its own smallest divisor except for 1).

## Examples

```ocaml
prime 2  = true
prime 3  = true
prime 4  = false
prime 17 = true
```
