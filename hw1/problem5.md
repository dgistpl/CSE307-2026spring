# Problem 5: `max : int list -> int` and `min : int list -> int`

**Homework 1 - CSE307, Spring 2026**

## Description

Write two functions

```
max : int list -> int
min : int list -> int
```

that find maximum and minimum elements of a given list, respectively.

## Examples

```ocaml
max [1;3;5;2] = 5
min [1;3;2]   = 1
```
